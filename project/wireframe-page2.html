Real-World Problem

Have you ever attempted to transcribe a video? There are tools such as oTranscribe that provides a user-friendly platform to perform transcription for media files, although the process remains slow and tedious. Perhaps you would like to provide captions for your own video and or you are very interested in video content that is delivered in a language in which you do not possesses linguistic competencies.

Solution

Whisper.ai is your solution.The source language is detected by default, you have the option to indicate the original language as you enter the run command using Anaconda prompt to activate a local environment. If this is not enough, you can use this value to simultaneously perform a translation of the transcription.
